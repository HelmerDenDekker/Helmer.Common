# Helmer.Common

## About

Common, shared code for dotNet 10 projects

## Projects

### Helmer.Shared.Common

Contains classes and extensions that have no dependencies on any other Frameworks or libraries than .NET 10.

### Helmer.Shared.Tools

Contains tools for ASP.NET MVC or Api projects. Like SecurityHeaders Middleware, the ReactiveX
DistinctBehahaviourSubject.

### Helmer.Shared.Infra

Contains tools for data access with Entity Framework Core. Like the Repository pattern.
